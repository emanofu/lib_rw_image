名前
	lib_rw_image - 画像ファイル読み書き

バージョン
	1.0.0

環境
	Windows 7 Professional SP1
	Microsoft Visual C++ 2008
	等

解説
	C/C++プログラムから呼び出して使うライブラリーツールキット。

	src_common/		共通ユーティリティ
	src_master/		各書式をまとめて使う共通ユーティリティ
	src_pic/		Softimage .picファイルの読み書き
	src_sgi/		Silicon Graphics .sgi/.rgbファイルの読み書き
	src_tga/		Targa .tgaファイルの読み書き
	src_tif/		TIFF .tifファイルの読み書き(libtiffが必要)
	README.txt		このファイル

使い方
	それぞれのソースコードを見てください

使用許諾
	ファイル"LICENSE"を見てください。
